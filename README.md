## About

Official website of the IEEE Robotics and Automation Society (RAS) Student Chapter at Madhav Institute of Technology & Science (MITS), Gwalior — a Deemed University.

This is a fully responsive, multi-page static website built with modern web technologies and designed to professional standards.

## Tech Stack

- **HTML5** — Semantic markup with accessibility features
- **CSS3** — Custom Properties, Grid, Flexbox, CSS Columns, animations
- **Vanilla JavaScript** (ES6+) — No frameworks or libraries
- **Google Fonts** — Space Grotesk (headings), Inter (body)

## Features

- ✅ Fully responsive (mobile, tablet, laptop, desktop)
- ✅ Smooth scroll animations using Intersection Observer API
- ✅ Canvas-based particle system on hero section
- ✅ Image lightbox gallery with masonry layout
- ✅ Animated number counters
- ✅ Contact form with client-side validation
- ✅ Event search and category filtering
- ✅ SEO optimized with Open Graph meta tags
- ✅ Accessible (ARIA labels, keyboard navigation, semantic HTML)
- ✅ Reduced motion support
- ✅ Page loading animation
- ✅ Back to top button

## Pages

| Page | Description |

## Getting Started

1. Clone or download the repository
2. Open `index.html` in any modern browser
3. Or use VS Code with Live Server extension for development

> No build step, no dependencies, no package manager required.
