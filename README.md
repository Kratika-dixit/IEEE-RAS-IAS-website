# IEEE RAS & IEEE IAS Website Foundation

This repository contains a modular, scalable static website foundation for the IEEE RAS and IEEE IAS Student Chapters.

## Structure

- `index.html` for the shared home page foundation
- `ras.html` for the RAS chapter page foundation
- `ias.html` for the IAS chapter page foundation
- `assets/css/` for modular stylesheets
- `assets/js/` for modular JavaScript files
- `assets/images/` for future page-specific images
- `assets/logos/` and `assets/icons/` for brand assets

## Notes

- No frameworks or build tools are required.
- The structure is GitHub Pages compatible.
- Developers can work independently on page sections while keeping shared infrastructure intact.
